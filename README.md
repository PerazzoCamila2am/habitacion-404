# La Habitación 404

Proyecto final de la materia Desarrollo y Arquitectura Web.

## Descripción

La Habitación 404 es un juego web tipo escape room. El jugador deberá resolver distintos puzzles, administrar su tiempo y utilizar pistas para escapar antes de que el sistema se reinicie.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES5
- LocalStorage
- Flexbox
